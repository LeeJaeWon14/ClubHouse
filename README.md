# ClubHouse
Fifa Online 4 support app
피파온라인4 API를 활용한 개임 전적 및 선수 거래 내역 조회 App.
</br>
[넥슨 개발자 센터 가이드라인](https://developers.nexon.com/fifaonline4/apiList)

2. Retrofit2
3. ViewModel
4. Glide
5. Room
